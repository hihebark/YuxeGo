YuxeGo
======

Exemple:
--------

`./YuxeGo -u https://www.youtube.com/watch?v=XXXXXXXXXXX`

convert to mp3:

`./YuxeGo -u https://www.youtube.com/watch?v=XXXXXXXXXXX -mp3`

```
  .----.
t(.___.t) - Yuxe
  `----
	0.2.0-dev

Usage of ./YuxeGo:
  -mp3
    	Convert to mp3 format if set.
  -o string
    	Output Folder. (default "Downloads/YuxeGo/")
  -q string
    	Quality: small (240p),
		medium (360p),
		large (480p),
		hd720 (720p),
		hd1080 (1080p).
  -u string
    	URL for the Youtube video.
```
